# cheskygo.com

My poetry + photography site. Poems are written in Google Docs and published
automatically — drop a doc in the Drive folder, it appears here.

## What's what

- `index.html` — the landing page + list of every poem.
  *Regenerated automatically when a new poem syncs. I don't edit this by hand.*
- `style.css` — the whole design (colors, fonts, layout).
  *This is the one file I edit to change how the site looks.*
- `poems/` — one HTML page per poem. *Generated automatically.*
- `assets/` — the header image for each poem. *Uploaded automatically.*

## How a poem gets here

1. Write the poem in Google Docs (image on top, title = file name).
2. Move the doc into the synced Drive folder.
3. Within ~15 min it's live on the site. No copy, no paste.

## Hosting

GitHub (this repo) → Netlify (builds & serves) → cheskygo.com
